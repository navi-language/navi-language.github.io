# 赋值

与大多数编程语言一样，使用 `=` 来给变量赋值。

```nvs
let a = 10 + 20;
// a = 30

a = a + 1;
// a = 31
```

我们也可以使用 `+=`、`-=`、`*=`、`/=` 来简化赋值操作。

```nvs
let a = 10;

a += 1;
// a = 11

a -= 1;
// a = 10

a *= 2;
// a = 20

a /= 2;
// a = 10
```

当你第一次声明一个变量时，你不能再次使用 `let` 来声明同一个变量，编译器会报错。

```nvs
let a = 10;

let a = 20;
// Compile error
```
